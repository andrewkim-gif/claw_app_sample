# 🐾 CrossClaw Sample Counter

CrossClaw OS 앱 개발 샘플 — SDK 기능 데모용 카운터 앱.

## Features

- **Storage API**: 카운터 값 + 방문 횟수 영속 저장
- **Notifications API**: 10의 배수 도달 시 알림
- **Window API**: 윈도우 타이틀 동적 변경
- **Lifecycle API**: 앱 시작/종료 이벤트 처리

## Install in CrossClaw OS

App Store → URL 입력 → 이 리포지토리 URL 붙여넣기

## Structure

```
crossclaw.manifest.json  — 앱 메타데이터 (필수)
index.html               — 엔트리 포인트
app.js                   — 앱 로직 + SDK 사용
style.css                — 스타일
```
